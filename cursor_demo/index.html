Custom Cursors

Today, I will show you how to change the normal cursor for your website into a custom one!

Let's first take a look at the simplest way

To make a custom cursor for the entire web page, you would use an image. This would require making a class and then making a CSS entry that looks like this:

            
                .module {
                    cursor: url('path-to-image.png'), auto;	
                  }

This is how the cursor for this web page is a planet, but here's the specific code:

            
                body {
                    cursor: url("../images/planet.png"), auto;	
                  }

Applying the CSS to the body tag makes the cursor change for the whole page.

But what if you want to change the cursor only when hovering over one element?

It would look a little something like this:

How are you feeling today?

The code for this Happy/Sad example is as follows (but without the first left angle brackets of each line):

            
                h1>How are you feeling today?
                input class="hide" type="checkbox" title="Happy" name="happy">
                label for="happy" class="option-happy">Happy
                
                input class="hide" type="checkbox" title="Sad" name="sad">
                label for="sad" class="option-sad">Sad
                
                input class="hide" type="checkbox" title="Meh" name="meh">
                label for="meh" class="option-meh">Meh

And this is what the CSS looks like:

            
                .option-happy {
                    cursor: url("https://s3-us-west-2.amazonaws.com/s.cdpn.io/9632/happy.png"), auto;
                  }
                
                  .option-sad {
                    cursor: url("../images/sad.png"), auto;
                  }
                  
                  .option-meh {
                    cursor: url("https://s3-us-west-2.amazonaws.com/s.cdpn.io/9632/meh.png"), auto;
                  }

This differs from the cursor for the whole page as these images were just connected to a class that would connect to a specific element
